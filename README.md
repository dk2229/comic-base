# Comic Base

漫画管理Webアプリケーション

## 概要
職業訓練校での開発演習で作成した、漫画情報を管理できるWebアプリケーションです。
HUNTER×HUNTERなどの漫画作品の情報、キャラクター、レビューを管理し、クイズ機能も楽しめます。

## 開発期間
2025年8月15日 - 9月11日（約1ヶ月）

## 使用技術
- Java
- PostgreSQL
- Servlet/JSP
- HTML/CSS
- MVCアーキテクチャ
- DAOパターン

## 主な機能
1. **ユーザー管理** - 会員登録・ログイン・編集・削除
2. **漫画管理** - 漫画情報のCRUD操作
3. **レビュー機能** - 5段階評価付き感想投稿
4. **掲示板機能** - 匿名での自由な書き込み・アンカー機能
5. **キャラクター図鑑** - キャラクター情報の管理
6. **マイページ** - 個人の投稿履歴管理
7. **クイズ機能** - HUNTER×HUNTER念能力読み方クイズ

## 技術的なこだわり
- 統一されたMVCパターンの実装
- 全機能でController→Service→DAOの一貫した流れ
- 複雑な日本語文字正規化処理（カタカナ・ひらがな統一）
- セキュリティを考慮したセッション管理

## 開発で学んだこと
- MVCアーキテクチャの実践的理解
- データベース設計とCRUD操作の実装
- エラーハンドリングとバリデーション
- 文字処理の複雑性と解決方法
- 統一されたコーディングパターンの重要性
